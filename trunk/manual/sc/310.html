3.10 Lysee Reference Manual - Language Reference - Directives

Includes {$SET}, {$SET::FILE}, {$SET::PATH} and {$INCLUDE}, directives are not casesensitive.
3.10.1 {$SET NAME_PATH VALUE}

Assign expanded VALUE to global NAME_PATH, likes: ${NAME_PATH} = @VALUE.

>>> {$SET SITE.TITLE Lysee Programming Language};
>>> = ${SITE.TITLE};
Lysee Programming Language
>>>
3.10.2 {$SET::FILE NAME_PATH FILE_NAME}

Expand FILE_NAME and assign to global NAME_PATH, likes: ${NAME_PATH} = FILE_NAME.fullFileName().excPD().

>>> {$SET::FILE FILE.WINI ${WINDIR}/win.ini};
>>> = ${FILE.WINI};
C:\WINDOWS\win.ini
>>>
3.10.3{$SET::PATH NAME_PATH PATH_NAME}

Expand PATH_NAME and assign to global NAME_PATH, likes: ${NAME_PATH} = PATHE_NAME.fullFileName().incPD().

>>> {$SET::PATH demo_path ${kndir}/demos};
>>> = ${demo_path};
C:\lysee\demos\
>>>
3.10.4 {$INCLUDE FILE_NAME}

Include file to current script, same as statement: incldue FILE_NAME;

>>> {$INCLUDE ${demo_path}eval.ls};
Input expression to evaluate or "quit" to leave!

lysee> 78 * 34
78 * 34 ==> 2652
lysee> quit
quit ==> bye-bye!
>>>
