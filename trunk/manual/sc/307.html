运算符按优先级排列分为以下几种:

NEG  : -   | NOT : not | BNOT: ~ | FMT : @  |          |        |        |         |          | 单目  : 高

7.3. 字符串连接

Lysee 使用加号“+”进行字符串连接。当加号的操作数中有一个是字符串时另一个操作数自动转换为字符串并完成连接。

>>> = "hello" + "world";
helloworld
>>> = "TIME: " + now();
TIME: Fri, 18 Apr 2008 18:57:52 GMT
>>>
7.4. 加倍连接

Lysee 使用乘号“*”进行字符串加倍连接。当乘号左侧是字符串，右侧是整数时，运算结果是加倍连接后的新串，否则结果是 nil。

>>> = "Aa" * 4;
AaAaAaAa
>>> = "Cc" * 1.42;

>>>
7.5. 左移插值

当左移运算符“<<”左侧是对象类型时，运算结果是对象自身，左移运算符尝试将右侧的值增加到左侧的对象中。

>>> cout << "I love " << "coding" << "!";
I love coding!
>>> = varlist() << 1 << 2 << "hello" << 4;
[1,2,hello,4]
>>> = strlist() << "hello" << "world";
hello
world
>>>

说明：支持左移运算符插值的对象类型包括 stream、strlist 和 varlist。
7.6. 列表定义

符号“[”和“]”用于定义列表常量，“[]”等同于 varlist()。下面是等价的两段代码：

>>> a = [1, 2, 3];
>>> = a;
[1,2,3]
>>>

 

 >>> a = varlist();
>>> a.add(1);
>>> a.add(2);
>>> a.add(3);
>>> = a;
[1,2,3]
>>>
7.6. 哈希表定义

符号“{”和“}”用于定义哈希表常量，“{}”等同于 hashed()。下面是等价的两段代码：

>>> a = {"name": "libudi",
  >      "sex" : "male",
  >      "age" : 40};
>>> for s in a.keys do
  >   = s, a.read(s), eol;
  > end;
age 40
sex male
name libudi
>>>		 >>> a = hashed();
>>> a.name = "libudi";
>>> a.sex = "male";
>>> a.age = 40;
>>> for s in a.keys do
  >   = s, a.read(s), eol;
  > end;
age 40
sex male
name libudi
>>>
