3.语言参考 - 1.构成

正文将介绍 Lysee 的保留字、标识符以及数字、字符串等的表达方法。
2.1 有效字符

Lysee 程序代码由以下字符或字符对构成：

名称 范围 用途
letter
 a..z A..Z _ 标识符、关键字
digit
 0..9 数字、标识符
char
 + - * / \ = < > [ ] . , ( ) : ^ @ { } $ # ` 运算符、界符、注释等
char pair // /* */ == != >= <= 运算符、注释等
space 空格、回车、换行、Tab、... 分隔符
2.2. 注释

Lysee 注释分行注释和块注释两种：

类型 开始标记 结束标记 示例
行注释
 # 所在行末 # This is a shell comment
行注释
 // 所在行末 // This a normal line comment
块注释
 /* */ /* This is a block comment */
2.3. 标识符

标识符用于表示 Lysee 程序中变量、常量、函数、类、模块等所有需要通过名称访问的对象。Lysee 标识符必须由英文字母、数字和下划线组成，且第一个字符必须是英文字母或下划线：

* 标识符 原因
正确
 JAMES_is_007  
正确
 _007  
错误
 007_is_JAMES 第一个字符是数字
错误 JAMES is 007 中间有空格

Lysee 标识符长度没有限制，建议尽可能简洁明了，不宜过长。

Lysee 标识符区分大小写，abc、Abc、ABc 和 ABC 就是四个完全不同的标识符。
2.4. 保留字

Lysee 保留字都是标准小写的标识符，在整个语言层面都是固定不变的，不允许被重复定义。

>>> = ${keywords};
__clen__,and,as,bool,break,case,char,class,const,continue,def,do,else,elseif,end
,endinc,except,false,finally,float,for,if,import,in,include,int,is,like,main,nil
,not,object,or,recall,repeat,return,string,super,switch,sys,then,this,time,true,
try,type,until,variant,void,while
>>>
2.5. 数字

数字分整数、浮点数、货币三种，默认为10进制数字，整数还可有16进制（0x）、8进制（0o）和2进制（0b）三种不同的表达方法，浮点数可以使用科学计数法：

类型 10进制 16进制 8进制 2进制 科学计数法
int
 2008 0x07D8 0o3730 0b11111011000 -
double 2008.4 - - - 2.0084e3
money
 100.0712 - - - -

布尔型（bool）属于特殊的整数类型，值只有真（1）、假（0）两个。
2.6. 字符和字符串

Lysee 字符为单字节的 ANSI 字符（0..255），在程序中必须包裹在配对的单引号中间，如‘a’和‘+’。

一些无法通过键盘录入的字符可以按‘\xHH’的格式转义表达，其中 HH 代表字符的16进制值，比如‘4’转义后为\x34’，一些常用的字符还有其专用的转义写法：

字符 16进制转义 ASCII 值 说明
\0
 \x00 0 字符串的结束标志
\a \x07 7 BELL
\b \x08 8 BACKSPACE
\e \x1B 27 ESCAPE
\f \x0C 12 FORMFEED
\n \x0A 10 NEWLINE
\r \x0D 13 CARRIGE RETURN
\t \x09 9 TAB
\v \x0B 11 VERTICAL TAB
\\ \x5C 92 转义符
\' \x27 39 单引号
\" \x22 34 双引号

字符串是以‘\0’结尾的字符序列，Lysee 字符串有三种表达形式且都允许跨行：

形式 开始标记 结束标记 示例 转义符 说明
1
 " " "kite\'s mother is
            jessy" 		有 遇见字符 \ 时进行字符转义处理
2
 R" " R"kite's mother is
            jessy" 		无 字符 \ 不再是转义符
3 """ """ """kite's mother is
            jessy"""		 无 开始和结束标记是连续的3个双引号
字符 \ 不再是转义符

第三种形式的字符串在找到配对的结束标记后将继续延伸截止到第一个不是双引号的字符：

>>> = """How many charactors in this string?"""""""";
How many charactors in this string?"""""
>>>
