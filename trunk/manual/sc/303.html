3.语言参考 - 3.变量
3.1 什么是变量

变量是指具有一定类型并经过命名的内存地址，Lysee 程序中只允许在函数内部定义临时的局部变量。
3.2. 局部变量

Lysee 通过赋值语句定义局部变量，可以为变量明确指定各种类型，缺省为 variant 类型：

>>> def get_min_value |v1, v2|
  >   if v1 > v2 then
  >     temp = v1; # define temp, type variant
  >     v1 = v2;
  >     v2 = temp;
  >   end
  >   return v1;
  > end;
>>> = get_min_value(10, 4);
4
>>>
3.3. 形参变量

形参变量指的是函数的参数（形参）。
