3.4 Lysee Reference Manual - Language Reference - Constants
3.4.1 What is constant value?

Contant is the readonly value.
3.4.2 Simple Constants

Contant values of builtin simple types:

Type Example Constant Values
bool true, false
char 'A', '\0', '\x34'
float 3.1415926, 0.618
int -843, 0, 1, 2008
string "I like coding"
3.4.3 Named Constants

Named constants are functions that can be run only once.

Named constants has 2 formats:

    1. const ConstantName [:DateType] = Expression ;
    2. const ConstantName [:DateType] = do Statements end

The first format is sugar for the second format:

    const ConstantName [:DateType] = do return Expression end

>>> const MAGIC:int = 3 * random(10);
>>> = dumpc(which("MAGIC"));
const MAGIC:int = do
    PUSH 3
    PUSH FUNC: sys::random
    PUSH 10
    CASK [2]
    CALC *
    RETURN LAST
end
>>> = MAGIC, MAGIC;
6 6
>>>
