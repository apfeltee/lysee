3.2 Lysee Reference Manual - Language Reference - Types

Objects in lysee has its own types.
3.2.1 Simple Types

There are 7 buildin simple types in kernel:

Type Description
bool Used as condition, value is true or false
char ANSI char
float 64BIT double
int int64
money currency in Pascal
string '\0' ended ANSI string
time TDateTime in Pascal
3.2.2 Function

Includes function, method and closure:

Type Description
function global function defined in module
method function defined in class
closure function that has no name
3.2.3 Class

Base element of OOP.
3.2.4 Variant

Variant can be any data type, depends on the time you check:

>>> v = "string"; // a string
>>> v = 18; // change to integer
>>> v = strlist(); // change to strlist
>>> v.add("hello world"); // add a line to strlist
>>> = v; // print strlist content
hello world
>>>
3.2.5 void

Used to define functions that has no result. value of void is nil.

Type Nil Value
bool false
char '\0'
float 0
int 0
money 0
string ""
time 0

You can check default value of data type with nil:

>>> strlist list;
>>> = "list has no value" if (list == nil);
list has no value
>>> = (10 == nil) ? 10 : 0;
0
>>>
