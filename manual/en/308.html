3.8 Lysee Reference Manual - Language Reference - Statements
3.8.1 Whate is statement?

Statement is the smallest standalone element of lysee script. Lysee program is formed by a sequence of one or more statements. Usually, Lysee statement ends with ';'.
3.8.2 Simple Statement

Includes assign value statement, expression statement and print statement.

3.8.2.1 Assign value to a variable or property:

>>> hs = hashed();
>>> hs.time = now();
>>> = hs.time;
Fri, 18 Apr 2008 19:14:20 GMT
>>>

3.8.2.2 Expression statement is a expression plus ';':

>>> (now() as string).saveToFile(R"c:\lysee\temp\now.txt");
>>> = R"c:\lysee\temp\now.txt".fileText();
Fri, 18 Apr 2008 19:18:49 GMT
>>>

3.8.2.3 Print statement starts with '=' and follows some values delimited by ','. ',' is deal as to ' ':

>>> = "I like coding with lysee";
I like coding with lysee
>>> = "Let's generate a magic value:", sys::random(1000);
Let's generate a magic value: 983
>>>
3.8.3 if statements

Symbol: if condition then
          [Statements]
        elseif condition then
          [Statements]
        else
          [Statements]
        end

>>> v = sys::random(10);
>>> if not v then  = "v=0";
  > elseif v <= 5 then  = "v <= 5";
  > else = "v > 5" end;
v <= 5
>>>
3.8.4 while statement

Syntax: while condition do
          [Statements]
        end

>>> v = sys::random(10);
>>> while v != 5 do
  >   = v, eol;
  >   v = sys::random(10);
  > end;
4
7
9
>>>
3.8.5 repeat until

Syntax: repeat
          [Statements]
        until condition;

>>> repeat
  >   v = sys::random(10);
  >   = v, eol;
  > until v == 5;
3
1
9
6
4
0
8
5
>>>

Statements in repeat until executes at least one time.
3.8.6 for statement

Syntax: for varb [in vargen [if condition]] do
          [Statements]
        end;

>>> for line in __envs__ if line like "^H" do
  >   = line.name, eol;
  > end;
HOMEDRIVE
HOMEPATH
>>>
3.8.7 switch statement

Syntax: switch expression
          case expr1: [Statements]
          ......
          case exprN: [Statements]
          else        [Statements]
        end

>>> switch sys::random(5)
  >   case  0: = 0;
  >   case  1: = 1;
  >   case  2: = 2;
  >   case  3: = 3;
  >   else     = 4;
  > end;
3
>>>

switch is deal as a special if, it can not be broken with keyword break.

3.8.9 import statement

Load lysee module into kernel.

Syntax: import module1[, module2, module_file3, module_file];

>>> import www;
>>> = www.funcs;
[def www::func_0001 ||,def www::wgets |URL|]
>>> import c:/lysee/demos/eval.ls;
>>> eval.main();
Input expression to evaluate or "quit" to leave!

lysee> 5*6
5*6 ==> 30
lysee> quit
quit ==> bye-bye!
>>>
3.8.10 include statement

Include specified file or part of file into current script.

Syntax: include [PART_NAME@]FILE_NAME;

>>> include c:/lysee/demos/eval.ls;
Input expression to evaluate or "quit" to leave!

lysee> 35*76
35*76 ==> 2660
lysee> quit
quit ==> bye-bye!
>>>
