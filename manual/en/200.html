2.3 Configuration

Add installation path to system search PATH:

Platform How to
Windows attch installation path to system environment variable PATH

Edit file lysee.config to configure lysee:
Variable Description Default Value
lse_tmpath temporary directory ${LYSEE}/temp
lse_search module search path ${LYSEE}/modules
lse_create_log if create log file? false

The following constants can be used in lysee.config:

2.5 Hello World

Let's print "Hello World":

D:\www.lysee.net>lysee
Lysee 0.1.6.387 - Copyright (C) 2003-2009 LiBUDI - www.lysee.net

           :: /C=CANCEL /F=FILE /Q=QUIT /R=RESTART ::

>>> = "Hello World";
Hello World
>>>

Is it simple?

Expl:   1. ">>>" prompts you enter lysee command or script.
    2. "=" means print.
    3. last ";" means end of input and lysee should execute the command or script.
2.6 Execute Script File

Save "= "hello world";" to file hw.ls, and run it with lysee:

D:\www.lysee.net>echo ="hello world"; > hw.ls
D:\www.lysee.net>lysee hw.ls
Hello World
D:\www.lysee.net>

[Expl]   1. Lysee script file's extension is ".ls".
    2. Lysee interpreter quits after execute the file.
    3. File consists of any lysee statements.
