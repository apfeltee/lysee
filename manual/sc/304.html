3.语言参考 - 4.常量
4.1 什么是常量

常量是指在程序代码中直接输入的数字、字符、字符串等具体的值，关键字 const 用于定义有名称的常量。
4.2. 数值常量

数值常量是在代码中直接输入的数字、字符、字符串等具体的值，对应着 Lysee 的简单数据类型：

类型 示例
bool 布尔值 true, false
char 字符 'A', '\0', '\x34'
float 浮点型 3.1415926, 0.618 （黄金分割）
int 整型 -843, 0, 1, 2008
string 字符串 "I like coding"
4.3. 命名常量

Lysee 中的命名常量是特殊的按名称调用函数，但求值表达式或语句仅执行一次。

定义命名常量的语法有以下两种:

    1. const ConstantName [:DateType] = Expression ;
    2. const ConstantName [:DateType] = do Statements end

第一种语法是第二种语法的简化形式，等同于：

    const ConstantName [:DateType] = do return Expression end

>>> const MAGIC:int = 3 * random(10);
>>> = dumpc(which("MAGIC"));
const MAGIC:int = do
    PUSH 3
    PUSH FUNC: sys::random
    PUSH 10
    CASK [2]
    CALC *
    RETURN LAST
end
>>> = MAGIC, MAGIC;
6 6
>>>
